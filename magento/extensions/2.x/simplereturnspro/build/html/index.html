What is Simple Returns Pro?¶

Simple Returns Pro is an advanced RMA extension intended for use with Magento Open Source. By extending the robust functionality of Simple Returns, Simple Returns Pro is able to provide an optimized RMA experience for both customers and administrators alike. In addition to the features provided by Simple Returns, Simple Returns Pro provides the following:

  1. Itemized returns

  2. Partial quantity returns

  3. Customizable product page badge

  4. Archive RMA requests

  5. Lock RMA requests

Before Getting Started¶

Prior to working with Simple Returns Pro, we highly recommend familiarizing yourself with Simple Returns, if you have not already done so. Simple Returns Pro is largely built upon the core functionality provided by Simple Returns, so it is helpful to understand the fundamentals before diving into the more advanced features provided by Simple Returns Pro.
